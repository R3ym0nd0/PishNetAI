What PhishNet AI checks

Instead of a long rating chart, here are the main signals the scanner reviews before showing a result.

Domain

Link structure

We look for unusual domain patterns, suspicious wording, shortened URLs, and misleading address structures.

Security

Trust signals

HTTPS, SSL status, and other website trust indicators are reviewed before the final result is shown.

Content

Page behavior

The scanner checks for phishing-style prompts, deceptive patterns, and risky page cues that push users to act fast.

Decision

Final guidance

You get a short summary, key indicators, and a recommendation so the result is easier to understand at a glance.

Quick anti-phishing reminders

Do This

Check the full domain name

Review the spelling carefully. Fake websites often use extra letters, missing letters, or unusual domain endings.

Watch Out

Pause when a message feels urgent

If a page or message pressures you to act immediately, stop first and verify it through an official source.

Before You Login

Inspect the URL before entering data

Never type passwords, student IDs, or personal details unless the address bar clearly matches the real website.
